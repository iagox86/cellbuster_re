Detector Settings Restore settings on this page

Enter the settings where prompted below. Click 'Save' to apply your changes or 'Back' to discard any changes. Hover over for help.

Detection Channel: Selecting an RF channelClick the select box to list the available RF detection channels and select one to view or to edit its parameters. Click the button 'Add' to add an RF channel with the displayed parameters - you can edit the parameters either before or after adding. Click 'Delete' to display a warning box, then click OK to confirm the deletion of the displayed channel.
Channel Name: Channel Detection: On Off Edit channel nameThe name of the channel must be unique and should describe the RF transmission you want to target. You can use up to 16 characters. Use the radio buttons to enable or disable RF detection in this channel. Click 'Save' to effect any change.
Detector RF Range:     to      MHz Setting the RF frequencySet the radio-frequency range to be monitored by the channel. Each value can be between 20MHz and 6020MHz. When you change the lower frequency (on left), the higher frequency will be set equal to it. Now edit the higher frequency (on right) to set a range of frequencies which cannot be greater than 100MHz. The minimum RF detection bandwidth (with both frequencies the same) is about 10MHz. This is how it works: if you select 900 to 930 MHz the device will in fact take separate measurements at 900, 910, 920 and 930MHz. Click 'Save' to effect any change.
Detection Method: Select detection methodFor most purposes select 'RF level alone'. The other options may help to discriminate between transmissions on the same frequency. For example the 'GSM modulation' option looks for the 216Hz pulse repetition rate characteristic of GSM. Click 'Save' to effect any change.
Sensitivity Threshold: MostLeast   dBr RF sensitivity thresholdMove the slider to either increase or decrease the detection sensitivity or type a value in the box. To respond only to stronger signals move the slider to the right. The value is calibrated in dB relative to an arbitrary origin. Click 'Save' to effect any change.
Audio Alert: Number of Repeats: Select an Audio AlertThe device is shipped with default audio alert options: these may be edited. Some audio alerts are recorded clips (wav files), others are 'jingles' (tone sequences). Select the number of times the sound is to be repeated per alert. Click 'Save' to effect any change.
Output Audio Alert: Select the Audio DestinationThe audio alert can be output on the device internal loudspeaker and/or to the 3.5mm jack at the rear. Click 'Save' to effect any change.
Visual Alert: Number of Repeats: Select a Visual AlertInstead or as well as an audio alert you can select a sequence of flashes on the high-brightness panel LED's. Select the number of times the sequence will be repeated per alert. Click 'Save' to effect any change.
Energise Relay Alert: On Off Enable the relay outputWhen 'On' each RF alert will energise the relay for a preset time. The relay has galvanomically isolated normally-open and normally-closed contacts available via the rear connector rated at 5A. Click 'Save' to effect any change.
