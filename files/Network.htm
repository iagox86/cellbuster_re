IP Addressing Method: IP Addressing MethodSelect 'DHCP' (the device will acquire an IP address from your DHCP server) or 'Manual' (you must enter a unique IP Address and default gateway e.g. as provided by your Network Administrator).
IP Address: Edit device IP AddressIf 'Manual' enter an IP address in the form xxx.xxx.xxx.xxx (e.g. 192.168.1.48). If you are unsure of what IP Address to use please contact your Network Administrator.
Subnet Mask: Edit device Subnet MaskIf 'Manual' enter an IP Submask in the form xxx.xxx.xxx.xxx (e.g. 255.255.255.0).
Default Gateway: Edit device Default GatewayIf 'Manual' enter the IP Address of the gateway (e.g. as used by the NTP feature) in the form xxx.xxx.xxx.xxx (e.g. 192.168.1.1).
DNS Server: Edit primary DNS ServerIf 'Manual' enter the IP Address of the first DNS server in the form xxx.xxx.xxx.xxx.
Alternate DNS Server: Edit alternate DNS ServerIf 'Manual' enter the IP Address of the second DNS server in the form xxx.xxx.xxx.xxx.
