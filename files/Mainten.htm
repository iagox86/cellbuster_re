Device Maintenance - Software Settings

Enter the settings where prompted or the various buttons for their actions. Hover over for help.

Backup Configuration Settings to PC: Backup configuration settingsClick the button to upload a backup file (backup.zip) of the current device configuration. The file will be deposited in your default download folder. It can only be 'opened' by Zone Protector and should not be edited.
Restore Configuration Settings from PC:
Restore configuration settingsYou can restore the configuration settings from a previously saved backup file. First click 'Select' and browse for the required file which must be named 'backup.zip' but may be located in any folder. Now click 'Restore' and, when prompted, click 'OK' if you are really sure. The operation will not affect any audio clips you have installed. After restoring the device must be reset to complete the process.
Firmware Version: 2.15 120530
Update Firmware:
Update firmware from fileYou must first obtain from your supplier a valid firmware update file. Click 'Select' and browse for this file. Finally click 'Update' to upload the file. Click 'OK' when you are prompted to confirm. The device status LED will start flashing amber to indicate the file upload which may take many seconds. Do not navigate away from this page until the upload is complete. When complete the device must be reset to complete the process.
Reset Device to Factory Defaults: Reset to factory defaultsAn alternative to restoring from backup is to reinstate factory defaults. After clicking this button click 'OK' when you are prompted to confirm. This operation will not affect any audio clips you have installed. You must reset the device to complete the process.
